# Space Invaders - Chrome Edition

A classic Space Invaders clone with retro arcade aesthetics, built as a browser game using HTML5 Canvas and Web Audio API.

## Play Now

**[Play Space Invaders](https://nikalston.github.io/space-invaders/)**

## Features

- Classic Space Invaders gameplay with 5 rows of alien invaders
- Three alien types: squid, crab, and octopus with animated sprites
- Destructible shields for cover
- Progressive difficulty across levels
- Local high score tracking
- Retro visual effects: CRT scanlines, film grain, and vignette
- Chrome-styled pixel art graphics
- Ominous synth bass soundtrack
- Spectacular explosion effects with screen shake

## Controls

| Key | Action |
|-----|--------|
| Left Arrow / A | Move left |
| Right Arrow / D | Move right |
| Space | Fire |
| P | Pause / Resume |
| Enter | Start / Restart |

## Scoring

| Alien | Points |
|-------|--------|
| Squid (top row) | 30 |
| Crab (middle rows) | 20 |
| Octopus (bottom rows) | 10 |
